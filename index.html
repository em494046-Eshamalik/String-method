toUpperCase()

Definition

Converts a string into ALL CAPITAL letters.

Why we use

To format text in uppercase like names or headings.

toLowerCase()

Definition

Converts a string into small letters.

Why we use

To compare text easily without case sensitivity.

trim()

Definition

Removes extra spaces from start and end of a string.

Why we use

To clean user input in forms.

slice()

Definition

Extracts part of a string and returns a new string.

Why we use

To get specific portion of text.

substring()

Definition

Returns part of string between two index positions.

Why we use

To extract a selected range of text.

replace()

Definition

Replaces a word or part of string with another value.

Why we use

To update or correct text.

includes()

Definition

Checks if a string contains a specific word or character.

Why we use

To search text inside a string.

indexOf()

Definition

Returns position (index) of first occurrence of a value.

Why we use

To find location of text in a string.

split()

Definition

Splits a string into an array based on separator.

Why we use

To break text into parts like words or data.

concat()

Definition

Joins two or more strings into one string.

Why we use

To combine multiple texts together.
